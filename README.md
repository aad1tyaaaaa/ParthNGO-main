# 🕊️ Parth Foundation (Path to Hope)

> **Architecting dignified futures through grassroots precision and radical transparency.**

Parth Organisation (Parth Foundation) is a non-profit organization based in Mumbai, India. Unlike traditional charitable organizations that provide temporary relief, we focus on architecting sustainable infrastructure in education, healthcare, and livelihood for communities that society has forgotten. We operate with radical transparency, measurable goals, and a commitment to the 2026 UN Sustainable Development Goals (SDGs).

---

## 🌍 Mission & Vision

**Our Mission:** To empower underserved communities through education, healthcare access, skill development, and sustainable livelihoods.  
**Our Vision:** A just and inclusive society where every individual has the opportunity to live with dignity and purpose.

### Core Values
* **People-Centered:** Communities are at the heart of every initiative.
* **Integrity:** Accountability and responsible stewardship of all resources.
* **Collaboration:** Working alongside partners, volunteers, and locals to create lasting change.

---

## 📊 Our Impact & Initiatives

We set hard numbers and track our progress transparently. To date, we have touched **10,000+ Lives** across **50+ Communities** with a **100% Dedicated Team**.

### Key Programs:
* 🧣 **Kambal Distribution:** Providing winter blankets to shield vulnerable populations from the harsh cold. *(Target: 2,000 blankets)*
* 🍲 **Nutritional Relief:** Eradicating hunger by providing daily nutritious meals to children. *(Target: 15,000 meals)*
* 🩺 **Medical Outreach:** Bringing vital health checkups and doctors directly to remote communities. *(Target: 15 health camps)*
* 🤝 **Cultural Unity:** Fostering community spirit through shared celebrations and events. *(Target: 20 functions)*

## 💻 Tech Stack

This platform is built using a modern, elite, production-ready architecture designed for high performance, deep accessibility, and Awwwards-level frontend interactions.

### Frontend
* **Core:** React 18.3, TypeScript 5.8
* **Build Tool:** Vite 5.4
* **Styling:** Tailwind CSS 3.4
* **Animations:** GSAP 3.13 (ScrollTrigger, complex timeline animations)
* **3D Graphics:** Three.js / React Three Fiber (Animated backgrounds)
* **Routing:** React Router DOM 6

### UI Components & Libraries
* **Primitives:** Radix UI
* **Design System:** Shadcn/UI (Customized Bento-box aesthetic)
* **Icons:** Lucide React & Phosphor Icons
* **Data Viz:** Recharts
* **Carousels:** Embla Carousel
* **Forms & SEO:** React Hook Form (Zod validation), React Helmet Async

### Backend, State & Services
* **Backend-as-a-Service:** Supabase (PostgreSQL, Authentication)
* **Payment Gateway:** Stripe API (Frictionless donation processing)
* **State Management:** TanStack React Query

### Development & Tooling
* **Runtime & Package Manager:** Bun
* **Linting & Formatting:** ESLint, PostCSS, Autoprefixer
